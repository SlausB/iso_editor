Конструктор(ы)

function Action.Action (   output:Output)

Конструктор. Устанавливает muteEnding в false.

Методы

function Action.Execute (   dispatcher:Dispatcher,
  consumeSeconds:Number 
)

Вызывается очередью, в которой хранится это задание.

Должно вызываться один раз в рамках одного цикла. Определяется в наследниках.

Аргументы:
dispatcherДиспетчер, в котором выполняется это действие.
consumeSecondsВремя в секундах, которое должно быть потреблено действием. Функция действия будет вызываться до тех пор, пока это время не будет полностью потреблено или пока действие не завершится.
Возвращает:
Сколько секунд действие потребило.

Переопределяется в ActionAsData и ActionAsTask.

Данные класса

var Action.abstractData [protected]

Значение, сформированное в последнем вызове Execute() если это ActionAsTask или просто хранящиеся данные если это ActionAsData .

var Action.muteEnding

Если установлено в true (по умолчанию false), то все попытки действия сообщить очереди что оно закончено (через Queue.ActionEnded() ) будут игнорироваться.

var Action.output

Средство вывода отладочной информации. Передаётся в конструкторе.

var Action.solidarity

Если установлено в true (по-умолчанию true), то действие не будет выполнять свой цикл если одно из его действий-параметров закончило своё выполнение (или было отмечено как действие, которое "хочет" завершиться).
