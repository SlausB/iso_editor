Подробное описание

Абстрактный тип переменной. Для возможности передачи параметров любого типа в ActionDesc . Обычно представляется как вещественное число.

ActionDesc представляет из себя функцию-действие, которое нужно совершить, и вектор параметров, которые являются типом вещественного числа. Но для некоторых действий нужны параметры других типов.
Хранит указатель на данные, которые интерпретируются по-умолчанию как вещественное число.

Методы

function AbstractData.AsFloat ( )

Получение данных как вещественное число. Если данные НЕ являются типом вещественного числа, то возвращает -1.

function AbstractData.Copy (   abstractData:AbstractData)

Копирование указанных данных.

Если оба объекта являются данными типа вещественного числа, то будет простое присвоение значение.
Если только текущие данные являются типом вещественного числа, то они будут удалены, а указатель на данные приравняеются у указателю на данные объекта abstractData . Если только указанные данные являются типом вещественного числа, то значение будет приравнено к данным типа вещественного числа указанного объекта ( abstractData ).

function AbstractData.GetData ( )

Получение текущих данных.

void* - это указатель на данные, которые используются во всех операторах (идёт работа как с double), а также эти данные будут удалены в деструкторе или в SetData() если они являются типом double, поэтому при удалении этим данных после получения возникнет ошибка сегментации.

function AbstractData.SetData (   data:*)

Установка новых данных. Предыдущие будут удалены.

data будут помечены как данные, не являющиеся типом вещественного числа.
Для установки переменной типа вещественного числа нужно использовать соответствующие операторы или установить данные посредством SetData() и указать что это вещественное число посредством DataIsInFloatTypeAgain().
