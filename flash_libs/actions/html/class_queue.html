function Queue.ActionEnded ( )

Текущее действие закончилось.

Используется в действиях для указания что оно закончилось.
Это лишь указание на то что действие закончилось. Перевод указателя Actions.Queue.currentAction и другие операции будут выполнены только когда управление вернётся функции Actions.Dispatcher.Live() (то есть когда завершится действие, вызвавшее эту функцию и/или действие, которое хранит внутри себя это действие как аргумент).

Действия.

var Queue.currentlyActiveAction

Действие, которое выполняется в данный момент.

По завершении выполнения действия или между циклами указатель становится неактуальным, поэтому используется только в функциях интерфейса выполняемого действия с очередью, в которой это действие находится. Устанавливается в Dispatcher.Live() .

var Queue.doneCycles

Сколько циклов выполняется текущее действие.

Увеличивается диспетчером в Live() .

var Queue.output [private]

Средство вывода отладочных сообщений. Передаётся в конструкторе.

var Queue.remainOnCurrentAction

true - после завершения текущего действия очередь сохранит это действия как снова "текущее". Сбрасывается после одного использования.

То есть после своего завершения действия снова начнёт выполняться с нуля. Сбрасывается после одного использования.

var Queue.wasActionEnded

Закончилось ли текущее действие в очереди.

Устанавливается в true в ActionEnded() .

var Queue.wasActionMarkedAsEnded

Пометило ли текущее действие себя как "оконченное".

Для того чтобы действие действительно закончилось нужно чтобы внутри него переменная muteEnding была равна false.
Будет установлено в true и в том случае, если действие закончилось.
