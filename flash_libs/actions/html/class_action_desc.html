function ActionDesc.Push (   args)

Добавление параметров, с которыми вызывать задание.

Аргументы:
argsПараметры, каждый из которых может иметь следующие типы:

  • Number или Int - параметр, который будет преобразован в AbstractData и передан в Task
  • ActionDesc - изменяемый параметр (тоже является действием)
  • AbstractData - данные произвольного типа. Обычно используется для передачи данным типа void*. В таком случае действие, которое считывает этот параметр, должно правильным образом использовать эти данные.
  • любое другое - преобразовывается в AbstractData

var ActionDesc.output [private]

Средство вывода отладочной информации. Передаётся в конструкторе.

var ActionDesc.solidarity

Завершить выполнение действия и оставшихся аргументов если один из аргументов завершил своё выполнение.

Если установлено в true (по-умолчанию true), то действие не будет выполнять свой цикл если одно из его действий-параметров закончило своё выполнение (при этом параметру должно быть разрешено завершать действие (см. Action.muteEnding ).
Перед вызовом самого действия его параметры выполняются слева-направо, поэтому после того как один из параметров завершит своё выполнение, то (если solidarity = true) остальные параметры, также как и это действие, выполняться не будут.
